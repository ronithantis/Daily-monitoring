# מערכת ניהול טיפול יומי בקשישים

אפליקציית ווב לניהול ומעקב אחר טיפול יומי בקשישים וסיעודיים.

## איך להשתמש

פשוט פתח את `index.html` בדפדפן, או העלה ל-GitHub Pages.

## תכונות
- מאגר 150 תרופות מסווגות
- מעקב רחצה, ארוחות, שתייה, צרכים, חיתולים
- פעילויות גופניות וקוגניטיביות
- מעקב מצב רוח ותסמיני דמנציה
- דוחות להדפסה
- אחסון מקומי (localStorage)
